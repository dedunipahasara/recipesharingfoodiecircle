# 🍳 FoodieCircle

FoodieCircle is a modern recipe sharing platform where users can create, manage, and explore recipes easily with a clean and responsive UI.

---

## ✨ Features

- 🔐 User Authentication (Login / Signup)
- 🧾 Create, update, delete recipes
- 🔍 Search & filter recipes by category
- 🖼️ Image upload support
- ❤️ Favorite recipes system
- 🔔 Toast notifications for actions
- 📱 Fully responsive design (mobile + desktop)

---

## 🛠️ Tech Stack

- React
- Vite
- Zustand
- Tailwind CSS
- DaisyUI
- Axios
- React Toastify
